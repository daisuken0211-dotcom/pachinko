# Orb Cascade - Prototype

概要:
- シンプルなブラウザ向けプロトタイプ（HTML5 Canvas + 自前物理）。
- 仕様 v1.0 に基づくコア挙動（発射 → 反射/ゾーン → ポータル → エナジー → 抽選 → フロー）を実装。

動作確認方法:
1. ファイル一式（index.html, main.js, config.json）を同じフォルダに保存します。
2. ローカルでそのフォルダを HTTP サーバから提供してブラウザで開くことを推奨します。
   - 例（簡易）: `npx http-server` または `python -m http.server 8000`
3. ブラウザで `http://localhost:8000/index.html` を開く。

操作:
- タッチ/マウス: 画面中でドラッグ（下→上）して狙いをつけ、指を離して発射。
- PC: ドラッグ＆リリース、または Space キーで簡易発射。
- 画面下の "Fire" ボタンで中間の発射も可能。

調整:
- 全チューニングパラメータは `config.json` にまとめています（ショット数、重力、速度、スコア、抽選確率など）。
- シード固定での再現はコンソールの `OC.restart(seed)` で可能です（例: `OC.restart(12345)`）。

保存:
- ハイスコアは `localStorage` に保存されます（キー: `oc_highscore`）。

注意:
- 本プロトタイプは仕様に沿った最小実装です。演出・音・UI の洗練、細かい物理チューニング、エフェクトやアクセシビリティ対応は別途強化できます。
- 実機の用語や演出に類する表現は使用していません（仕様の禁止事項に準拠）。
